# Tourist Office Management System

Full-stack web application developed as part of a university Software Engineering course.

The system allows tourism offices to create and manage travel packages, handle reservations and communicate with customers.

## Features
- User registration and login
- Travel package creation and management
- Search and filtering of travel destinations
- Reservation workflow with approval process
- Email notifications for booking confirmation
- Admin / organizer management interface

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express
- GraphQL
- TypeScript

Database:
- MySQL / MariaDB

## Project Structure
backend/
- src/ → TypeScript backend source code
- database/ → database schema and SQL files
- package.json → Node.js dependencies
- tsconfig.json → TypeScript configuration

frontend/
- components/ → UI components
- code/ → frontend JavaScript logic
- styles/ → CSS styling
- HTML pages (index.html, login.html, bookings.html, etc.)
